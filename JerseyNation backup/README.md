# JerseyNation

JerseyNation is a Razor Pages web application designed as a World Cup 2026 jersey store prototype for the USF Application Development course.

## Overview
This project was created as an academic prototype of an online jersey store focused on national teams participating in the 2026 World Cup. The website combines product presentation, basic e-commerce structure, and personalized recommendations based on local user behavior.

## Current Features
- Homepage introducing the project idea, purpose, and key features
- Shop page with highlighted jerseys
- Promotions and special jerseys sections
- Personalized "For You" recommendations based on local user behavior
- Match-based recommendations
- Product details page structure, currently non-functional
- Cart page structure, currently non-functional
- Privacy page structure, currently non-functional

## Project Structure
- `Pages/` - site pages
- `Models/` - data models
- `Data/` - mock product and match data
- `wwwroot/` - static files such as CSS, JavaScript, and images

## How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run the application
4. Open the Shop page and test the recommendation behavior

## Current Status
This project is a functional prototype. Some pages, such as Product Details, Cart, and All Jerseys, are currently simplified placeholders prepared for future expansion.
