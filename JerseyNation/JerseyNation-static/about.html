About JerseyNation

JerseyNation is an e-commerce website focused on national team jerseys and related football products. This project was built as an individual assignment and combines storefront design, product organization, data modeling, and interactive pages for analytics and business presentation.

Below you can see the same business-block ERD preview from the full data-model page. When you click any ERD block, the full ERD page opens with more details from the data dictionary, relationships, and API integration notes.

Background

My first HTML certification dates back to 1996, using a program called HotDog. Later, through my background in Advertising, I built many web pages and earned certifications in e-commerce development with ASP pages using Drumbeat, as well as in advanced dynamic pages built with Flash, which was widely used at the time for interactivity.

Although I have not worked professionally as a web developer for more than 15 years, I remained close to the field by leading large technology teams that delivered software projects in cybersecurity, marketing, sales, debt collection and customer service. Even when I was not coding directly, I stayed closely connected to web development and systems design.

This Assignment

This website was developed as an individual project. In addition to the storefront and content pages, the project includes a structured logical data model that supports customers, orders, products, payments, shipping, suppliers, and marketing activities.

Business-block ERD View

This page includes an interactive image of the ERD Diagram. Even for a relatively small website, many tables are needed to manage the system properly and support safe, organized data administration. For that reason, the ERD is presented in business blocks, and the detailed description of each block appears below together with the Data Dictionary, which can also be accessed from the dedicated button on the full ERD page.

Hover a block to preview richer table detail. Click any block in the diagram or use the buttons above to open the full ERD page, the data dictionary, or the API details page.
